🌟 Project Overview

The "UPI Fraud Detection System" is an advanced, real-world machine learning application designed to identify and prevent fraudulent digital transactions in real time.

With the rapid growth of digital payments, fraud detection has become a critical challenge. This system leverages **AI-driven intelligence** to analyze transaction patterns and classify them as **Fraudulent 🚨 or Genuine ✅**.

🔐 Fraud Detection Strategy
The system identifies fraud based on:

✔ Unusual transaction amount
✔ Unknown device or location
✔ High-frequency transactions
✔ Abnormal behavioral patterns
